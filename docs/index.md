## 기술 스택 한눈에 보기

| 도구 | 역할 | 쉽게 말하면 |
|------|------|------------|
| **Next.js** | 웹사이트 프레임워크 | 사이트의 뼈대 |
| **React** | 화면 만들기 | 버튼, 카드 같은 부품 |
| **TypeScript** | 프로그래밍 언어 | 실수를 줄여주는 자바스크립트 |
| **Tailwind CSS** | 디자인 도구 | 코드로 스타일 입히기 |
| **Drizzle ORM** | 데이터베이스 연결 | 데이터를 쉽게 다루기 |
| **Vercel** | 배포 서비스 | 사이트를 인터넷에 올리기 |
