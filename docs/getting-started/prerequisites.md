# 사전 준비 — 필요한 프로그램 설치하기

안녕하세요! HRA 웹사이트 개발팀에 합류하신 것을 환영해요. 본격적으로 코드를 만지기 전에, 우리 컴퓨터에 개발 도구들을 먼저 설치해야 해요.

---

### 1. GitHub 계정 만들기
GitHub은 우리가 짠 코드를 저장하고 공유하는 '클라우드 저장소'예요.

1. [github.com](https://github.com/)에 접속해서 **Sign up** 버튼을 눌러요.
2. 안내에 따라 이메일 주소, 비밀번호, 닉네임을 설정해요.
3. 입력한 이메일로 인증 메일이 오면, 링크를 클릭해서 인증을 마쳐요.

!!! success "콜라보레이터 초대 수락"
    계정을 만들었다면 관리자에게 알려주세요. 초대 메일이 오면 반드시 메일함에서 **Accept Invitation**을 눌러야 프로젝트에 참여할 수 있어요.

---

### 2. Node.js 설치
Node.js는 우리가 만든 웹사이트가 돌아가게 해주는 '엔진' 같은 거예요.

1. [nodejs.org](https://nodejs.org/ko/)에 접속해요.
2. **LTS**(Long Term Support)라고 적힌 버전을 다운로드해서 설치해요.

=== "확인 방법"
    터미널을 열고 다음 명령어를 입력했을 때 숫자가 나오면 성공이에요!
    ```bash title="Node 버전 확인"
    node -v
    ```

---

### 3. VS Code (Visual Studio Code) 설치
메모장 대신 코드를 더 편하게 쓸 수 있는 '스마트 메모장'이에요.

1. [code.visualstudio.com](https://code.visualstudio.com/)에서 운영체제에 맞는 버전을 설치해요.
2. 설치가 끝나면 실행하고, 왼쪽 메뉴의 사각형 아이콘(Extensions)을 눌러 다음 프로그램을 설치하는 걸 추천해요:
    - **Korean Language Pack**: 메뉴를 한국어로 바꿔줘요.
    - **Tailwind CSS IntelliSense**: 스타일링 코드를 쓸 때 자동 완성을 도와줘요.
    - **ESLint**: 코드에 오타나 실수가 있으면 빨간 줄로 알려줘요.

---

### 4. Git 설치하기
Git은 코드의 '변경 이력'을 관리해주는 프로그램이에요.

=== "Mac"
    이미 설치되어 있을 확률이 높아요. 터미널에서 `git --version`을 입력해보세요. 만약 "명령어를 찾을 수 없다"고 나오면, 팝업창의 **설치** 버튼을 눌러 Xcode Command Line Tools를 설치하면 돼요.

=== "Windows"
    [git-scm.com](https://git-scm.com/)에서 다운로드해서 설치해요. 설치 과정 중 설정은 기본값(Next)으로만 넘겨도 충분해요.

=== "확인 방법"
    터미널에서 다음을 입력했을 때 버전 숫자가 나오면 돼요.
    ```bash title="Git 버전 확인"
    git --version
    ```

---

### 5. 터미널(Terminal)이 뭔가요?
검은 화면에 직접 글자를 타이핑해서 컴퓨터에게 명령을 내리는 창이에요.

- **Mac**: '터미널' 앱을 실행하세요. (Spotlight에서 검색 가능)
- **Windows**: 'PowerShell' 또는 'Git Bash'를 실행하세요.

앞으로 나오는 명령어들은 이 터미널에 한 줄씩 복사해서 붙여넣고 엔터를 치면 돼요!
