# AI 도구로 기여하기 — OpenCode, Codex, Claude 활용법

코딩이 처음이라도 걱정하지 마세요. HRA 웹사이트 프로젝트는 AI 도구를 적극적으로 활용하여 누구나 쉽게 기여할 수 있도록 설계되었습니다. 한국어로 원하는 기능을 설명하기만 하면 AI가 코드를 대신 작성해 줍니다.

## 1. AI 코딩 도구가 뭔가요?

AI 코딩 도구는 여러분이 한국어로 "이런 기능을 만들어줘"라고 말하면, 프로젝트의 기존 코드를 분석해서 직접 코드를 짜주거나 수정할 부분을 알려주는 똑똑한 비서입니다.

| 도구 | 특징 | 무료 여부 | 난이도 |
|------|------|----------|--------|
| **GitHub Copilot Agent** | 이슈만 쓰면 AI가 PR까지 자동 생성 | 학생 무료 (구독 필요) | ⭐ (가장 쉬움) |
| **OpenCode (터미널 AI)** | 터미널에서 대화하며 파일 자동 수정 | 학생 무료 (구독 필요) | ⭐⭐ |
| **Claude (웹/앱)** | 대화하듯 코드 질문 및 생성 | 무료/유료 | ⭐ |

---

## 2. GitHub Copilot Coding Agent로 기여하기

가장 쉬운 방법입니다. 코딩 도구를 설치할 필요 없이 GitHub 웹사이트에서 글만 쓰면 됩니다.

### 사용 단계

1.  **GitHub 저장소 접속**: [github.com/yeongseon/hra-website](https://github.com/yeongseon/hra-website)에 접속해요.
2.  **이슈(Issue) 생성**: `Issues` 탭 클릭 → `New Issue` 버튼을 눌러요.
3.  **내용 작성**: 제목과 본문에 원하는 내용을 구체적으로 적어주세요.
    ```markdown title="이슈 작성 예시"
    ## 요청사항
    교수진 페이지(src/app/(public)/faculty/page.tsx)에서 각 교수님 이름 옆에 프로필 사진이 표시되도록 수정해주세요.
    
    ## 참고사항
    - 이미지는 public/images/faculty/ 폴더에 있어요.
    - 기존 코드 패턴을 따라주세요.
    - UI 텍스트는 한국어로 작성해주세요.
    ```
4.  **Copilot 호출**: 이슈 라벨에 `copilot`을 추가하거나, 댓글로 `@copilot`을 언급해요.
5.  **자동 작업**: Copilot이 자동으로 브랜치를 만들고 코드를 수정한 뒤 **Pull Request(PR)**를 생성해요.
6.  **검토 및 완료**: 생성된 PR의 코드를 확인하고, 문제가 없으면 `Merge` 버튼을 눌러 반영해요!

!!! tip "꿀팁"
    파일 경로(예: `src/app/page.tsx`)를 정확히 적어줄수록 AI가 더 정확하게 일해요.

## 3. OpenCode로 기여하기

OpenCode는 여러분의 터미널(검은 창)에서 AI와 대화하며 실시간으로 파일을 수정하는 도구입니다.

### 설치 및 실행

```bash title="터미널에서 입력하세요"
# Mac 사용 시
brew install opencode

# 또는 npm으로 설치
npm install -g opencode
```

### 사용법

1.  **프로젝트 폴더**에서 터미널을 열고 `opencode`를 입력해요.
2.  **한국어로 명령**하세요. AI가 파일을 직접 읽고 수정해 줍니다.
    - "footer.tsx 파일에서 저작권 문구를 'Copyright © HRA'로 바꿔줘"
    - "교수진 페이지에 새 교수 '홍길동'을 추가해줘"
    - "이 에러 메시지가 뭔지 설명해줘: [에러 메시지 복사/붙여넣기]"
3.  **결과 확인**: AI가 작업을 마치면 `npm run dev`를 입력해 웹사이트에서 바뀐 모습을 확인해요.
4.  **저장 및 제출**:
    ```bash title="변경사항 저장하기"
    git add .
    git commit -m "교수진 페이지 수정"
    git push origin feature/내-이름-작업
    ```

## 4. Claude로 기여하기

Claude는 웹사이트(claude.ai)에서 대화하듯 질문하고 코드 답변을 받는 도구입니다. 복잡한 로직을 이해하거나 새로운 페이지를 설계할 때 유용해요.

### 활용 시나리오

*   **코드 이해하기**: 코드를 복사해서 "이 코드가 뭘 하는 건지 초보자도 알기 쉽게 설명해줘"라고 물어보세요.
*   **코드 생성하기**: "Next.js로 공지사항 목록 페이지를 만들어줘. Tailwind CSS를 사용하고 표 형식으로 보여줘"라고 요청하세요.
*   **에러 해결**: 터미널에 뜬 빨간색 에러 메시지를 복사해서 "이 에러가 왜 나는 거야? 어떻게 고쳐?"라고 물어보세요.

!!! info "Claude에게 질문할 때"
    "HRA 웹사이트는 **Next.js + TypeScript + Tailwind CSS**를 사용해요"라고 배경 정보를 먼저 알려주면 훨씬 정확한 답을 얻을 수 있어요.

## 5. AI 도구 활용 시 주의사항

!!! warning "꼭 읽어주세요"
    *   **검토 필수**: AI가 만든 코드가 항상 완벽하진 않아요. 반드시 직접 작동하는지 확인하세요.
    *   **한국어 UI**: 사용자에게 보이는 모든 텍스트는 반드시 한국어로 작성해야 해요. (영어 금지!)
    *   **보안 주의**: `.env` 파일이나 비밀번호, API 키 같은 민감한 정보를 AI에게 보내지 마세요.
    *   **타입 우회 금지**: `as any`나 `@ts-ignore`처럼 에러를 억지로 무시하는 코드는 사용하지 마세요.

## 6. 어떤 도구를 쓰면 좋을까요?

| 내 상태 | 추천 도구 |
|---------|----------|
| 코딩을 아예 모르고, 게시판에 글 쓰듯 기여하고 싶어요 | **GitHub Copilot Agent** |
| 내 컴퓨터 터미널에서 AI랑 대화하며 바로 고치고 싶어요 | **OpenCode** |
| 특정 기능이 어떻게 돌아가는지 공부하며 질문하고 싶어요 | **Claude** |
| 아주 간단한 문구나 색상만 고칠 거예요 | **아무거나 (Claude가 가장 친절함)** |
