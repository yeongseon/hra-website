# 보고서 작성 가이드

본 가이드는 회원 자료실의 [보고서 템플릿](/member/templates)을 채울 때 참고하는 문서입니다.
**템플릿 본문에는 별도의 안내문이 들어가지 않으므로**, 무엇을 어떻게 쓸지는 이 가이드에서 확인해주세요.

## 1. 작성 절차

1. 분야에 맞는 양식을 선택해 [보고서 템플릿](/member/templates) 페이지에서 PDF로 다운로드
2. Markdown 본문 작성
3. 조원 검토
4. 지도교수 컨펌
5. 최종 PDF 변환 후 제출

## 2. 양식 선택 기준

| 분야 | 사용할 양식 | 슬러그 |
| --- | --- | --- |
| 경영서 | 경영서 보고서 양식 | `management-book` |
| 고전명작 | 고전명작 보고서 양식 | `classic-book` |
| 기업실무·한국경제사 | 기업실무·한국경제사 보고서 양식 | `business-practice` |

경영서·고전명작 양식은 동일한 5장 구조(`1. 책 소개`, `2. 저자 소개`, `3.`, `4.`, `5.`)를 사용합니다.
기업실무·한국경제사 양식은 빈 5장 구조(`1.`, `2.`, `3.`, `4.`, `5.`)를 사용하고 발표 정보 표에 조원 목록을 포함합니다.

## 3. 발표 정보 표 작성 방법

| 항목 | 작성 방법 |
| --- | --- |
| 소속 | `HRA 19기 1조`처럼 기수와 조 번호를 명확히 표기 |
| 조원 (기업실무 양식) | `학과 학년 이름` 형식으로 줄바꿈하여 모두 기재 |
| 발표자 | 학과·학년·이름 |
| 담당교수 | 지도교수 성함 + `교수님` |
| 발표일 | `2026. 03. 07. (토)` 형식 |

## 4. 각 장 작성 방법 (경영서·고전명작)

### 1. 책 소개

- 책의 주제
- 선정 이유
- 핵심 문제의식
- 현재 사회 또는 기업과의 연결점

### 2. 저자 소개

- 저자의 주요 경력
- 학문적·실무적 배경
- 대표 저작
- 저자가 가진 관점/사상

### 3 ~ 5.

조에서 정한 분석 주제를 자유롭게 구성합니다.
가능하면 각 장은 다음 4요소를 포함합니다.

- 제목
- 그림 (필요 시)
- 내용
- 출처

## 5. 각 장 작성 방법 (기업실무·한국경제사)

빈 5장 구조이므로 조에서 다음을 자유롭게 구성합니다.

- 주제 선정 배경
- 시대적·산업적 배경
- 사례/데이터 분석
- 평가
- 시사점

## 6. 기존 HWP 서식의 Markdown 변환 기준

기존 HWP에는 다음과 같은 서식 지시(예: 맑은 고딕 32pt 진하게)가 있었습니다.
Markdown 템플릿에서는 이런 지시를 본문에 노출하지 않고, 아래 표 기준으로 변환합니다.
항목 기호는 대한민국 공문서 표준(「행정 효율과 협업 촉진에 관한 규정 시행규칙」)을 따릅니다.

| 기존 HWP 서식 | 공문서 항목 기호 | Markdown 기준 |
| --- | --- | --- |
| 제목 32pt 진하게 | — | `# 제목` |
| 1단위 타이틀 12pt 진하게 | `1.` | `## 1. 제목` |
| 2단위 타이틀 12pt 진하게 | `가.` | `### 가. 제목` |
| 3단위 타이틀 11pt 진하게 | `1)` | `#### 1) 제목` |
| 4단위 타이틀 10pt | `가)` | `##### 가) 제목` |
| 5단위 타이틀 10pt | `(1)` | `###### (1) 제목` |
| 6단위 이하 | `(가)`, `①`, `㉮` | 본문 인라인 표기 |
| 본문 10pt | — | 일반 문단 |
| 각주 및 참고자료 8pt | — | `## 참고자료` |

> 참고: PDF 출력 시 본문 글꼴은 Pretendard / Noto Sans KR을 사용합니다.
> 쪽 번호는 자동으로 제거됩니다.

## 7. 그림·이미지

- 본문에 이미지를 넣을 때는 `![설명](https://...)` 문법을 사용합니다.
- 허용된 출처: Vercel Blob 업로드 이미지, 동일 출처(`/...`) 상대 경로
- 외부 도메인 이미지는 보안상 차단되므로 다운로드 후 자료실을 통해 업로드해야 합니다.

## 8. 참고자료

- 마지막 장 다음에 `## 참고자료` 섹션을 추가합니다.
- KCI(한국학술지인용색인) 표기법을 따릅니다.

## 9. 검토 체크리스트

- [ ] 발표 정보 표 모든 항목 작성
- [ ] 5장 목차 누락 없음
- [ ] 그림·표 깨짐 없음
- [ ] 출처·참고자료 명시
- [ ] 맞춤법 검사

## 10. 자주 묻는 질문

**Q. Markdown이 처음입니다.**
[Markdown 문법 가이드](/member/guides/markdown-guide)에서 기본 문법을 확인하세요.

**Q. 이미지가 표시되지 않아요.**
허용된 도메인(Vercel Blob, 동일 출처)에서만 표시됩니다.

**Q. PDF 다운로드는 어디서?**
[보고서 템플릿](/member/templates) 상세 페이지의 "PDF로 다운로드" 버튼을 클릭하세요.

**Q. 제출 방법은?**
[제출 안내](/member/guides/submission-guide) 문서를 확인하세요.
