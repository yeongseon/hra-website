# Markdown 문법 가이드

본 문서는 HRA 보고서 작성 시 사용하는 Markdown 문법을 정리한 가이드입니다.
보고서 양식의 단계 구분은 [보고서 작성 가이드](/member/guides/report-writing-guide)에서 정의한
**대한민국 공문서 표준 항목 기호**(`1. → 가. → 1) → 가) → (1)`)를 따릅니다.

## 1. 제목 (Heading)

| 단계 | 문법 | 예시 |
| --- | --- | --- |
| 1단계 | `#` | `# 제목` |
| 2단계 | `##` | `## 1. 책 소개` |
| 3단계 | `###` | `### 가. 저자의 경력` |
| 4단계 | `####` | `#### 1) 학문적 배경` |
| 5단계 | `#####` | `##### 가) 대표 저작` |
| 6단계 | `######` | `###### (1) 인용 사례` |

## 2. 글자 강조

| 표현 | 문법 | 결과 |
| --- | --- | --- |
| 굵게 | `**굵게**` | **굵게** |
| 기울임 | `*기울임*` | *기울임* |
| 굵은 기울임 | `***둘다***` | ***둘다*** |
| 취소선 | `~~취소~~` | ~~취소~~ |
| 인라인 코드 | `` `코드` `` | `코드` |

## 3. 목록

### 가. 순서 없는 목록

```markdown
- 항목 1
- 항목 2
  - 하위 항목
```

### 나. 순서 있는 목록

```markdown
1. 첫 번째
2. 두 번째
   1. 하위 항목
```

### 다. 체크리스트

```markdown
- [ ] 미완료
- [x] 완료
```

## 4. 인용문

```markdown
> 인용할 내용을 작성합니다.
> 여러 줄 인용도 가능합니다.
```

## 5. 표

```markdown
| 항목 | 내용 |
| --- | --- |
| 소속 | HRA 19기 1조 |
| 발표일 | 2026. 03. 07. (토) |
```

> 셀 안에서 줄바꿈이 필요한 경우 `<br>` 태그를 사용합니다(예: 조원 명단).

## 6. 링크

```markdown
[표시할 글자](https://hra.example.com)
[보고서 양식](/member/templates)
```

## 7. 이미지

```markdown
![대체 텍스트](https://...)
```

- 허용 출처: Vercel Blob 업로드 이미지, 동일 출처(`/...`) 상대 경로
- 외부 도메인 이미지는 보안 정책상 차단됩니다. 자료실에 업로드 후 해당 URL을 사용하세요.

## 8. 코드 블록

````markdown
```
일반 코드
```

```python
print("언어 지정 가능")
```
````

## 9. 구분선

```markdown
---
```

## 10. 자주 쓰는 패턴

### 가. 발표 정보 표

```markdown
| 항목 | 내용 |
| --- | --- |
| 소속 | HRA 19기 1조 |
| 발표자 | 경영학과 3학년 홍길동 |
| 담당교수 | 김ㅇㅇ 교수님 |
| 발표일 | 2026. 03. 07. (토) |
```

### 나. 출처 표기

```markdown
**출처**: 저자명, 『도서명』, 출판사, 출판연도, 인용 페이지.
```

## 11. 주의 사항

- 한 줄 띄어쓰기로 문단을 구분합니다.
- 제목 위·아래에는 빈 줄을 넣어 가독성을 확보합니다.
- HTML 태그는 일부만 허용됩니다(`<br>` 등). 보안 정책상 `<script>`, `<iframe>` 등은 차단됩니다.
- 자세한 단계 구분 규칙은 [보고서 작성 가이드 §6](/member/guides/report-writing-guide#6-기존-hwp-서식의-markdown-변환-기준)을 참고하세요.
